Introducción

Este dashboard conecta el sistema de facturación InterFuerza con el programa de lealtad Devotio Rewards. Cada vez que un cliente compra en la tienda, el sistema detecta la factura automáticamente y acredita el cashback en su tarjeta de puntos — sin que el cajero tenga que hacer nada.

El dashboard le permite al equipo monitorear las acumulaciones en tiempo real, procesar canjes cuando un cliente quiere usar su saldo, y gestionar los casos donde el sistema no pudo identificar al cliente automáticamente.

Roles en la plataforma

👔
Gerente / Dueño
  • Ver estadísticas del negocio
  • Ajustar el % de cashback
  • Resolver clientes sin match
  • Revisar el historial completo
🛒
Cajero
  • Buscar a un cliente
  • Procesar canjes de cashback
  • Ver el balance disponible
Sobre la acumulación automática El cashback se acredita solo — el cajero no necesita hacer nada en el dashboard después de una venta. La acumulación ocurre automáticamente cada 20 minutos.

Acceso al Sistema

El dashboard funciona en cualquier navegador moderno (Chrome, Firefox, Safari, Edge). No requiere instalación.

1
Abrir el navegador

Ir a la dirección URL del dashboard que le proporcionó el administrador.

2
Ingresar el correo y contraseña

Usar las credenciales asignadas por el administrador. La contraseña se configura la primera vez mediante un correo de invitación.

3
Click en "Ingresar"

El sistema lo redirige automáticamente a la Vista General.

Sesión activa El sistema mantiene la sesión abierta en el navegador. No es necesario ingresar las credenciales cada vez, salvo que cierre sesión manualmente o cambie de dispositivo.

Cerrar sesión

En la esquina inferior izquierda del menú lateral aparece el botón Cerrar sesión. Úselo al terminar de usar el dashboard en una computadora compartida.

Pasos para procesar un canje

1
Obtener el contacto del cliente

Pedirle al cliente su número de teléfono o correo electrónico.

2
Buscar en el dashboard

Escribir el teléfono (con prefijo 507), correo, o nombre del cliente en el campo de búsqueda y presionar Buscar.

3
Verificar el saldo

El sistema muestra el nombre del cliente y el saldo disponible en su tarjeta. Confirmar con el cliente cuánto desea canjear.

4
Click en "Canjear"

Se abre un panel con el saldo completo pre-llenado. Si el cliente quiere canjear un monto menor, escribir el monto parcial.

5
Confirmar el canje

Click en Confirmar. El sistema descuenta el saldo de la tarjeta Devotio del cliente y muestra una confirmación verde.

6
Aplicar el descuento en la factura

El monto canjeado debe aplicarse manualmente como descuento en la factura de InterFuerza.

Importante El canje descuenta el saldo de Devotio inmediatamente. Si se confirma por error, contactar al administrador para revertirlo desde el panel de Devotio.
5

Historial de Transacciones

Muestra todas las facturas procesadas por el sistema, con la opción de filtrar por fecha, sucursal o estado. Se accede desde el menú lateral en Transacciones.

Estados de una transacción

Procesado

El cashback fue acreditado correctamente en la tarjeta Devotio del cliente.

Pendiente

El sistema no pudo identificar al cliente en Devotio. Requiere resolución manual desde Configuración.

Para el gerente Este historial sirve como registro de auditoría. Puede exportar la vista del navegador o tomar capturas para reportes mensuales.
6

Configuración

Acceso exclusivo para gerentes. Se accede desde el menú lateral en Configuración.

Cambiar el porcentaje de cashback

1
Ir a Configuración en el menú lateral
2
Editar el campo "Porcentaje de cashback"

Escribir el nuevo porcentaje (ej. 3 para un 3%).

3
Click en "Guardar"

El nuevo porcentaje aplica a partir del próximo ciclo de sincronización (máximo 20 minutos).

Atención El cambio aplica solo a facturas futuras. Las facturas ya procesadas mantienen el porcentaje con el que fueron calculadas.

Estado de la sincronización

En la misma pantalla aparece la fecha y hora de la última sincronización con InterFuerza, y la frecuencia configurada (cada 20 minutos).

7

Resolver Transacciones Pendientes

Cuando el sistema no puede identificar a un cliente en Devotio — porque el número de teléfono o correo no coincide exactamente — la factura queda marcada como Pendiente. El gerente puede resolverla manualmente desde Configuración.

1
Ir a Configuración → "Transacciones pendientes"

La lista muestra nombre del cliente de InterFuerza, número de factura y monto.

2
Click en "Resolver →" en la fila correspondiente

Se abre un panel con los datos del cliente según InterFuerza.

3
Buscar al cliente en Devotio

Escribir el nombre, teléfono o correo del cliente en el campo de búsqueda. Aparecerán los resultados de Devotio.

4
Seleccionar el cliente correcto

Confirmar que el nombre y contacto coinciden con el cliente de InterFuerza.

5
Click en "Confirmar ✓"

El cashback se acredita en la tarjeta seleccionada, la fila desaparece del panel y la transacción queda como Procesado.

Cuándo ocurren los pendientes Sucede cuando el cliente tiene un número diferente en InterFuerza y en Devotio, o si aún no tiene tarjeta Devotio activa. El sistema intenta buscar también por correo electrónico antes de marcarla como pendiente.

Preguntas Frecuentes

¿Cuánto tiempo tarda en aparecer el cashback después de una venta?

El sistema sincroniza con InterFuerza cada 20 minutos. El cashback aparece en la tarjeta del cliente en ese lapso máximo.

Un cliente dice que no ve su cashback en Devotio. ¿Qué hago?

Primero, verificar en Historial de Transacciones que la factura fue procesada. Si aparece como Pendiente, ir a Configuración y resolver la transacción manualmente. Si no aparece, puede ser que aún no ha pasado el ciclo de 20 minutos.

Confirmé un canje por error. ¿Se puede revertir?

El sistema no tiene reversión automática de canjes. Contactar al administrador para revertirlo directamente desde el panel de Devotio Rewards.

¿Qué pasa si busco a un cliente y no aparece en Canjear Cashback?

El cliente puede no estar registrado en Devotio Rewards todavía, o puede tener un teléfono o correo diferente. Intentar buscar por nombre. Si tampoco aparece, el cliente debe registrarse en Devotio primero.

¿El cliente necesita mostrar algo para canjear?

No necesariamente. Con el número de teléfono o correo es suficiente para identificarlo en el sistema.

¿Puedo cambiar el porcentaje de cashback en cualquier momento?

Sí, desde Configuración. El cambio aplica a facturas futuras a partir del próximo ciclo. Solo los gerentes tienen acceso a esta función.

¿El sistema funciona si el internet cae momentáneamente?

Las facturas de InterFuerza se procesan en el siguiente ciclo de sincronización. Si el internet cae, el sistema recupera las facturas pendientes en la próxima ejecución y las procesa normalmente.
